#  AI Interview Prep Platform  
### *AI-Powered Interview Simulation, DSA Practice & ATS Resume Builder*

An intelligent full-stack platform that helps users prepare for interviews through AI-generated questions, skill analysis, company-based DSA practice, feedback system, and ATS resume generation.
Live Demo: https://ai-interview-prep-omega-coral.vercel.app

Backend API: https://ai-interview-prep-ffjr.onrender.com

---

##  Overview

AI Interview Prep Platform is a complete career preparation system that combines:
- Interview simulation
- Skill-based learning guidance
- Company-specific DSA practice
- AI feedback engine
- ATS resume builder

It transforms traditional interview preparation into a structured AI-driven experience.

---

##  Problem

Job seekers struggle with:
- Unstructured interview preparation  
- No real-time feedback  
- Lack of role-specific guidance  
- Difficulty in DSA interview preparation  
- Poor ATS resume formatting  

---

##  Solution

A unified AI platform that guides users through:
**Learn → Practice → Test → Improve → Get Job Ready**

---

##  Features

### AI Interview Generator
- Select role, difficulty, and number of questions
- AI generates role-specific interview questions
- Adaptive and structured interview flow

---

###  Skill Intelligence System
- Enter any job role (e.g. Full Stack Developer)
- Get required skills breakdown
- Learning roadmap suggestions

---

###  DSA + Company Interview Practice
- Choose company (Google, Amazon, etc.)
- Select topic, level, and question count
- Solve coding problems with test cases
- AI evaluates your solution instantly
- Feedback on logic, complexity & approach

---

###  AI Feedback Engine
- Code analysis in real-time
- Suggests optimal solutions
- Identifies mistakes & improvements
- Enhances problem-solving approach

---

###  ATS Resume Builder
- Input personal details, skills, projects
- AI generates ATS-friendly resume
- Professional formatting
- Download as PDF

---

###  Authentication System
- Signup / Login system
- JWT authentication
- Password hashing using bcrypt
- Forgot password with OTP verification
- Redis used for session & OTP handling

---
